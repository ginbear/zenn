# Zenn Articles

https://zenn.dev/ginbear の GitHub 連携リポジトリ

## セットアップ

```bash
npm install
```

## 執筆

### 新しい記事を作成

```bash
npx zenn new:article
```

### プレビュー

```bash
npx zenn preview
```

http://localhost:8000 で確認

### 公開

1. 記事の `published: true` に変更
2. コミット & プッシュで自動デプロイ
